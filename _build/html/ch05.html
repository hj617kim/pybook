5. 기본자료형¶

자료형Data Type은 컴파일러나 인터프리터에게 데이터를 어떻게 사용할 것인지 알려주는 데이터 속성 중 하나다.
이번 장에서는 파이썬 프로그래밍 언어에서 사용되는 기본 자료형을 간단히 살펴보자.

  • 정수 int

  • 부동소수점 float

  • 불리언 bool

  • 문자열 str

5.1. 정수 int¶

일반적으로 알고 있는 정수(자연수, 0, 음의 정수)들의 자료형이다. 정수형으로 나타내면 덧셈, 뺄셈, 곱셈, 나눗셈 등의 기본 연산이 가능하다.
예를 들어, …, -3, -2, -1, 0, 1, 2, 3, …은 모두 정수형(int)이다.

5.2. 부동 소수점 float¶

부동소수점은 원래 실수를 컴퓨터에서 다루기 위해 개발되었으나 실제로는 유리수 일부만을 다룬다. 무리수인 원주율 \(\pi\)의 경우에도 컴퓨터의 한계로 인해 소수점 이하 적당한 자리에서 끊어서 사용한다. 예를 들어, 1.2, 0.333, -1.2, -3.7680, 4.0 등은 모두 부동소수점형(float)이다.

5.3. 문자열 str ¶

문자들을 나열한 값들을 일컫는 자료형으로 작은 따옴표(') 또는 큰 따옴표(")를 사용한다.

'Hello, World'

'Hello, World'

"Hello, World"

'Hello, World'

Example 5.1

Hello, "World"를 보이는 그대로 출력하는 코드를 작성하여라.

문자열 내부에서 큰 따옴표를 사용하고 싶을 때는 작은 따옴표로 문자열을 감싸면 된다.

print('Hello, "World"')

Hello, "World"

백슬래시(\, 한글 키보드에서는 원화기호￦를 사용)를 사용하여 문자열 내부에서 사용되는 따옴표의 특수 역할을 해제escape할 수 있다.

print("Hello, \"World\"")

Hello, "World"

반대로 문자열 내부에서 작은 따옴표를 사용하고 싶을 때는 큰 따옴표로 문자열을 감싸면 된다.

print("I'm a student")

I'm a student

여러 줄로 이루어진 문자열은 삼중 따옴표(''' ''' 또는 """ """)를 사용할 수 있다.

print('''
Hello,
 World
''')

Hello,
 World

print("""
Hello,
 World
""")

Hello,
 World

5.3.2. whitespace¶

whitespace는 화면상으로 아무것도 표시되지 않는 공백 문자를 말한다.

string.whitespace는 공백으로 간주하는 문자를 포함하는 문자열로, 여기에는 스페이스space, 탭tab, 줄변경line feed or new line, 캐리지 리턴carriage return, 수직탭vertical tab, 폼피드form feed 문자가 포함된다.

import string
string.whitespace

' \t\n\r\x0b\x0c'

기호

설명

공백 space

\t

탭 tab

\n

줄변경 line feed or new line

\r

캐리지 리턴 carriage return

\v 또는 \x0b

수직탭 vertical tab

\f 또는 \x0c

폼피드form feed

 
print('1\n2\t3')

1
2	3

print('123\b4')

1234

  • 캐리지 리턴(\r)은 그 줄의 맨앞으로 이동시키는 특수 문자이다.

print('1234\r56')

1234
56

  • \v 또는 \x0b는 수직탭이고, \f 또는 \x0c는 다음 페이지로 이동하는 것을 의미한다.

문자열 ''과 문자의 ' ' 차이첨

빈 문자열empty string ''는 아무런 문자도 포함하지 않았기 때문에 문자열의 길이 즉, 문자열에 포함된 문자의 개수가 0이다.
반면 ' '는 눈에 보이지는 않지만 공백 문제 하나를 포함하는 문자열이며 길이가 1이다.

5.3.3. 문자열에 특수 문자 활용하기¶

백슬래시(￦), 줄바꾸기(￦n), 탭(￦t) 등은 문자열에 사용될 경우 특수한 기능을 갖는다. 그리고 백슬래시(\)기호를 붙여서 특수 기능을 해제escape할 수 있다.

print("Hello\\n World")

Hello\n World

print("Hello\\t World")

Hello\t World

print("Good\\night")

Good\night

연속된 백슬래시 두 개를 출력하려면 아래와 같이 해야 한다.

print("\\\\")

\\

주의

아래와 같이 코드를 작성하면 오류가 발생하므로 주의해야 한다.

>>> print("\\\")
  File "/tmp/ipykernel_76/2337434698.py", line 1
    print("\\\")
                ^
SyntaxError: EOL while scanning string literal

참고

순수 문자열 ‘가공되지 않은’의 의미를 갖는 ‘raw’ 단어의 첫 글자인 ‘r’을 문자열 앞에 두면 특수 기능이 사라진다.

print(r"Hello\n World")

Hello\n World

print(r"Hello\t World")

Hello\t World

print(r"Hello\ World")

Hello\ World

5.3.5. 문자열과 print() 함수¶

그 동안 출력을 위해 print() 함수를 사용하였다. 예를 들어, Hello, python을 출력하고 싶다면, 아래와 같이 코드를 작성하면 된다.

print('Hello, python')

Hello, python

이제 print() 함수에 대해서 조금 더 살펴보자.

  • 큰 따옴표(또는 작은 따옴표)로 둘러싸인 문자열을 연속해서 사용하면 문자열에서 +연산을 한 것과 동일한 결과를 출력해준다.

print("Hello ""World")
print('Hello ''World')
print('Hello '"World")
print("Hello " + "World")

Hello World
Hello World
Hello World
Hello World

  • 문자열 사이에 콤마(,)를 사용하면 띄어쓰기를 할 수 있다.

print("Hello", "World")

Hello World

  • 인자들 사이에 구분자를 넣고 싶다면, sep을 변경하면 된다. sep의 기본값은 공백이다.

print("010", "1234", "5678", sep = "-")
print("Hello", "World", sep = ", ")

010-1234-5678
Hello, World

  • 마지막에 출력할 문자를 변경하고 싶다면, end를 변경하면 된다. end의 기본값은 줄변경(\n)이다.

print("a")
print("b", end = "")
print("c", end = "\t")
print("d")

a
bc	d

5.3.6. 문자열 포매팅 String formatting¶

5.3.6.1. str.format()¶

format()메서드를 사용하면, 문자열의 중괄호({})가 format()의 인자로 변경된다. 예제와 함께 살펴보자.

'{}님, 안녕하세요.'.format('강현')

'강현님, 안녕하세요.'

name = '강현'
'{}님, 안녕하세요.'.format(name)

'강현님, 안녕하세요.'

age = 3
'강현이는 {}살이다.'.format(age)

'강현이는 3살이다.'

여러 개의 값을 변경할 때는 콤마(,)로 구분해서 적어준다.

name = '강현'
age = 3
'{}이는 {}살이다.'.format(name, age)

'강현이는 3살이다.'

인덱스 항목을 사용하여 넣어줄 위치를 지정할 수도 있다.

name1 = '강현'
name2 = '나영'
print('{}이와 {}이는 친구다.'.format(name1, name2))
print('{0}이와 {1}이는 친구다.'.format(name1, name2))
print('{1}이와 {0}이는 친구다.'.format(name1, name2))

강현이와 나영이는 친구다.
강현이와 나영이는 친구다.
나영이와 강현이는 친구다.

소수점 표현

콜론(:) 뒤에 소수점아래 몇 번째 자리까지 출력할지를 적어주면, 그 만큼을 보여준다.
예제와 함께 살펴보자.

  • .은 소수점을 의미하고, 소수점 뒤의 숫자는 소수점 뒤에 나올 숫자의 개수다.

num = 0.123456789
print('{0:.1f}'.format(num))
print('{0:.2f}'.format(num))
print('{0:.3f}'.format(num))
print('{0:.5f}'.format(num)) # 소수점 아래 여섯 번째 자리에서 반올림

0.1
0.12
0.123
0.12346

5.3.6.2. f-string¶

문자열 앞에 f 를 붙이면, 문자열 포매팅 기능을 사용할 수 있다.
예제와 함께 살펴보자.

name = '강현'
f'{name}님, 안녕하세요.'

'강현님, 안녕하세요.'

age = 3
f'강현이는 {age}살이다.'

'강현이는 3살이다.'

name = '강현'
age = 3
f'{name}이는 {age}살이다.'

'강현이는 3살이다.'

{}안에 변수와 수식(+, -, *, / 등)을 함께 사용하는 것도 가능하다.

name = '강현'
age = 3
f'{name}이의 동생은 {age - 2}살이다'

'강현이의 동생은 1살이다'

num = 0.123456789
print(f'{num:.1f}')
print(f'{num:.2f}')
print(f'{num:.3f}')

0.1
0.12
0.123

5.4. 불리언 bool ¶

참(True)과 거짓(False)를 나타내는 자료형이다. 파이썬에서 불리언 자료형은 int 형의 자식형 subtype이고, 대부분의 상황에서 True와 False는 각각 1과 0처럼 동작한다.

5.5. 형변환 함수¶

형변환 type casting은 자료형을 변환하는 것으로, 파이썬은 다양한 형변환 함수를 제공한다.

  • int() : 정수형으로 변환. 부동소수점을 정수로 변환할 때, 소수점 이하는 버려진다.

  • float() 부동소수점형으로 변환.

  • str() : 문자열형으로 변환.

  • bool() : 불리언형으로 변환

int(4.8)

4

int('5')

5

int(True)

1

float(3)

3.0

float('7.9')

7.9

str(6)

'6'

주의

int()함수의 인자로 문자열을 사용할 때는 그 모양이 정수모양이어야 하고, float() 함수의 인자로 문자열을 사용할 때는 그 모양이 정수 또는 부동소수점 모양이어야 한다.

>>> int('5.0')
ValueError                                Traceback (most recent call last)
/tmp/ipykernel_279/3485297474.py in <module>
----> 1 int('5.0')

ValueError: invalid literal for int() with base 10: '5.0'

>>> float('5GB')
ValueError                                Traceback (most recent call last)
/tmp/ipykernel_279/814975516.py in <module>
----> 1 float('5GB')

ValueError: could not convert string to float: '5GB'

참고

bool() 함수는 0, 0.0, ''(빈 문자열)처럼 0이거나 비어 있는 값을 인자로 사용하면 False를 반환한다.

print(bool(0))
print(bool(0.0))
print(bool(''))
print(bool(2))
print(bool(4.2))
print(bool('Hello'))

False
False
False
True
True
True

주의

변수를 형변환한다고 해서 변수에 할당된 값이 변하는 것은 아니다. 다만, 형변환한 값을 다른 변수에 저장해서 사용할 수는 있다.

basic_int = 2
print(float(basic_int))
print(type(basic_int))

2.0
<class 'int'>

float_basic_int = float(basic_int)
print(float_basic_int)
print(type(float_basic_int))

2.0
<class 'float'>
